This site itself. Built with Astro for static output, MDX for rich blog posts,
and Tailwind for styling. Content collections provide schema-validated frontmatter
so I get build-time errors instead of runtime surprises.

The goal was something fast, simple, and maintainable — no React, no client-side
JS unless needed.
